Mantis Roadmap

by Kenzaburo Ito (prescience@300baud.org)

Feature and Version Timetable

This is the Mantis feature roadmap. It lists all the currently planned features and details the schedule for release. These are not hard deadlines and things are subject to change. Please feel free to suggest features, contribute, or comment.

0.12.0

  • Logged in as notice on each page along with date/time
  • Limit bugnote delete access
  • Show the bugs that a user has reported, fixed, has assigned
  • Attach descriptions to bugs

0.13.0

  • Simpler custom header
  • Simpler custom footer
  • Move some global settings into the database
  • Resolution Dialog
  • Reopen Dialog

1.x.0

  • Email notification
  • Let users create their own accounts and have passwords emailed to them
  • Configuration Documentation
  • Attach files to bugs (file upload)
  • show longest time for a bug is open
  • show average length a bug is open
  • Priority support
  • Projection support
  • ETA support
  • Vote support
  • Add more fields for bug report details (CPU, RAM, etc)

2.x.0

  • Database abstraction layer
  • Improved style sheet support and conformance
  • PHPLIB/PEAR?
  • Session support
  • HTTP Authentication?
  • Multiple project support
  • Weekly report generation
  • Print friendly form
  • Put strings in one file for localization
  • Switch to unicode?
  • "I forgot my password" email (requires different password storage system
  • Security Audit
  • String/form handling Audit
  • IP Tracking / banning
  • Little glyphs and graphics for site
  • Bug Search
  • Allow users to specify required fields
  • Allow user to create own required fields
  • Interfaces for custom configuration

3.x.0

  • Rewrite using objects
